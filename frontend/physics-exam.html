1．关于运动和力的关系，下列说法正确的是（　　）

A．物体受到的合力为零时，一定处于静止状态B．物体的速度越大，所受的合力一定越大C．物体受到恒定的合力作用时，一定做匀变速运动D．物体做曲线运动时，合力方向一定与速度方向相同
▶ 查看答案与解析
【答案】C
A项，合力为零时物体静止或做匀速直线运动，不一定静止，错误。B项，速度大小与合力大小无直接关系，如匀速运动速度可能很大但合力为零，错误。C项，恒定合力则加速度恒定，即匀变速运动（包括匀加速和匀减速），正确。D项，曲线运动的合力方向与速度方向不在同一直线上，错误。

2．一个质量为 m 的物体从高为 h 的光滑斜面顶端由静止开始下滑，到达斜面底端时的速度大小为（斜面倾角为 θ）（　　）

A．√(2gh)B．√(2gh sin θ)C．√(2gh cos θ)D．2gh / sin θ
▶ 查看答案与解析
【答案】A

▶ 查看答案与解析

11．（10分）在"探究加速度与力、质量的关系"实验中，使用打点计时器和小车，已知小车质量 M=0.5 kg，砂和砂桶的总质量 m=0.1 kg，重力加速度 g=10 m/s²。

实验中得到一条纸带，测得计数点间距依次为：2.40 cm、2.80 cm、3.20 cm、3.60 cm、4.00 cm、4.40 cm（相邻计数点间有4个点未画出，打点周期 T₀ = 0.02 s）。

（1）打点计时器使用的电源为__________（填"交流"或"直流"）电源。

（2）利用逐差法计算小车的加速度 a = __________ m/s²。

（3）实验中认为砂和砂桶的重力等于小车受到的合力，这种近似的条件是什么？

（4）若实验中未平衡摩擦力或平衡摩擦力不足，画出的 a-F 图像会有什么特征？

▶ 查看答案与解析
【解析】
(1) 交流电源（打点计时器使用交流电源，频率50Hz）。

(2) 相邻计数点时间间隔 T = 5×0.02 = 0.1 s。
逐差法：a = [(s₄+s₅+s₆)-(s₁+s₂+s₃)]/(9T²)
= [(3.60+4.00+4.40)-(2.40+2.80+3.20)]×10⁻²/(9×0.01)
= [12.00-8.40]×10⁻²/0.09
= 3.60×10⁻²/0.09 = 0.40 m/s²

(4) 若未平衡摩擦力或平衡不足，a-F 图像在 F 轴上有一段截距（当 F 较小时加速度为零），即图像不过原点，需要施加一定的力才能使小车产生加速度。

13．（10分）如图所示，半径为 R 的光滑圆形轨道竖直放置，质量为 m 的小球从与圆心等高处的 A 点由静止释放，沿轨道运动到最低点 B。

（1）求小球到达 B 点时的速度大小。

（2）求小球在 B 点受到的轨道支持力。

（3）若小球到达 B 点后继续沿轨道向上运动，求小球能上升的最大高度。

▶ 查看答案与解析
【解析】
(1) 由 A 到 B，机械能守恒：mgR = (1/2)mv_B²
v_B = √(2gR)

(2) 在 B 点，合力提供向心力：N - mg = mv_B²/R
N = mg + m(2gR)/R = mg + 2mg = 3mg
支持力为 3mg，方向竖直向上。

(3) 小球过 B 点后沿轨道另一侧上滑，机械能守恒。设最高点高度为 h（从 B 点算起）：(1/2)mv_B² = mgh
h = v_B²/(2g) = 2gR/(2g) = R
即从 B 点上升 R 的高度，到达与 A 等高处（圆心高度）。这说明小球能上升的最大高度等于圆心高度（从最低点起算为 R，从释放点算则与释放点等高）。

14．（10分）如图所示，两平行金属板水平放置，上板带正电，下板带负电，板间电压为 U，板间距离为 d。一质量为 m、电荷量为 +q 的带电粒子从上板正中央的小孔以初速度 v₀ 竖直向下射入板间。不计粒子重力。

（1）求板间电场强度的大小和方向。

（2）求粒子到达下板时的速度大小。

（3）若粒子到达下板后又以原速率弹回（弹性碰撞），求粒子在两板间运动的时间。

▶ 查看答案与解析
【解析】
(1) E = U/d，方向由正极板指向负极板，即竖直向下。

(2) 粒子带正电，电场力方向向下，与运动方向相同，电场力做正功。
由动能定理：qU = (1/2)mv² - (1/2)mv₀²
v = √(v₀² + 2qU/m)

(3) 粒子到达下板后以原速率弹回（向上），加速度大小不变（方向向下，即减速上升）。
向下运动时间 t₁：由 d = v₀t₁ + (1/2)(qU/md)t₁²（匀加速）
或用 v² - v₀² = 2(qU/md)·d → v = √(v₀²+2qU/m)（已求）
t₁ = (v - v₀)/(qU/md) = md(v-v₀)/(qU)

弹回后向上运动到最高点（速度为0）：t₂ = v/(qU/md) = mdv/(qU)
总时间：t = t₁ + t₂（若要求往返总时间则继续计算）。
也可直接用 v = v₀ + at₁，a = qU/(md)，t₁ = (v-v₀)/a = md(v-v₀)/(qU)
弹回后上滑到停：t₂ = v/a = mdv/(qU)
总时间 = t₁ + t₂ = md(2v-v₀)/(qU)，其中 v = √(v₀²+2qU/m)。

15．（10分）如图所示，两根平行光滑金属导轨间距为 L，置于水平面上，导轨电阻不计。左端连接一个阻值为 R 的电阻，磁感应强度为 B 的匀强磁场垂直导轨平面向上。一根质量为 m、电阻为 r 的导体棒 cd 横放在导轨上，以初速度 v₀ 向右运动。不计摩擦。

（1）求导体棒中产生的感应电动势的表达式。

（2）求导体棒中的感应电流方向和安培力方向。

（3）求导体棒最终停止时通过电阻 R 的总电荷量。

▶ 查看答案与解析
【解析】
(1) 感应电动势 E = BLv，其中 v 为导体棒的瞬时速度。

(2) 由右手定则（或楞次定律），导体棒向右运动，磁通量增大，感应电流的磁场向下→电流方向为 d→c（在棒中从 d 到 c）。安培力方向由左手定则判断：F = BIL，方向向左（阻碍导体棒运动），与 v₀ 方向相反。

(3) 设导体棒从初速 v₀ 到停止的过程中，通过的总电荷量为 q。
对导体棒用动量定理：-BIL·Δt = 0 - mv₀
即 -BLq = -mv₀
q = mv₀/(BL)
（注意此处电流在变化，用微分形式：Σ(BIL·Δt) = BLΣ(I·Δt) = BLq）

16．（10分）如图所示，在光滑水平面上有三个质量分别为 m、m、2m 的小球 A、B、C，其中 A 以速度 v₀ 向右运动与静止的 B 发生弹性正碰，碰后 B 再与静止的 C 发生弹性正碰。

（1）求 A 与 B 碰撞后，A 和 B 的速度。

（2）求 B 与 C 碰撞后，B 和 C 的速度。

（3）若三个球的质量均相同（都为 m），A 以 v₀ 撞 B，B 再撞 C，求最终三个球的速度。

▶ 查看答案与解析
【解析】
(1) A(m, v₀) 与 B(m, 静止) 弹性碰撞：
质量相等的弹性正碰→交换速度
v_A' = 0，v_B' = v₀

(2) B(m, v₀) 与 C(2m, 静止) 弹性碰撞：
动量守恒：mv₀ = mv_B'' + 2mv_C''
动能守恒：(1/2)mv₀² = (1/2)mv_B''² + (1/2)(2m)v_C''²
解得：v_B'' = (m-2m)v₀/(m+2m) = -v₀/3（B 反弹）
v_C'' = 2mv₀/(m+2m) = 2v₀/3
验证：动量 m(-v₀/3)+2m(2v₀/3) = -mv₀/3+4mv₀/3 = mv₀ ✓
动能 (1/2)m(v₀/3)²+(1/2)(2m)(2v₀/3)² = mv₀²/18+4mv₀²/9 = mv₀²/18+8mv₀²/18 = mv₀²/2 ✓

(3) 三球质量均为 m：
A 撞 B：v_A'=0, v_B'=v₀（交换速度）
B 撞 C：v_B''=0, v_C''=v₀（交换速度）
最终：v_A=0, v_B=0, v_C=v₀
即速度从 A 依次"传递"给 C。
