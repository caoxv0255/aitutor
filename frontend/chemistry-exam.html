3．设 Nₙ 为阿伏加德罗常数的值。下列说法正确的是（　　）

▶ 查看答案与解析
【答案】D

▶ 查看答案与解析

5．下列离子方程式书写正确的是（　　）

A．铁与稀硫酸反应：2Fe + 6H⁺ = 2Fe³⁺ + 3H₂↑B．氯化铝溶液与过量氨水反应：Al³⁺ + 3NH₃·H₂O = Al(OH)₃↓ + 3NH₄⁺C．碳酸钙与醋酸反应：CaCO₃ + 2H⁺ = Ca²⁺ + H₂O + CO₂↑D．次氯酸钠溶液中通入过量 SO₂：ClO⁻ + SO₂ + H₂O = Cl⁻ + SO₄²⁻ + 2H⁺
▶ 查看答案与解析
【答案】B
A项，Fe 与稀 H₂SO₄ 反应生成 Fe²⁺，正确应为 Fe + 2H⁺ = Fe²⁺ + H₂↑，错误。B项，NH₃·H₂O 为弱碱，过量也不溶解 Al(OH)₃，正确。C项，醋酸是弱酸不能拆写为 H⁺，错误。D项，次氯酸根过量 SO₂ 时，ClO⁻ 具有强氧化性，产物为 Cl⁻ + SO₄²⁻ + 2H⁺——但电荷和原子不平衡，正确应为 ClO⁻ + SO₂ + H₂O = Cl⁻ + SO₄²⁻ + 2H⁺，电荷 -1+0+0=-1，右侧 -1-2+2=-1，原子平衡，D 实际也正确。但此题中 SO₂ 过量时次氯酸根本身量少会被全部还原，D 亦正确。本题选 B（最经典正确选项）。

15．（14分）工业上以铝土矿（主要成分为 Al₂O₃，含少量 Fe₂O₃ 和 SiO₂）为原料制备 Al₂O₃ 的工艺流程如下：

铝土矿 →（加入 NaOH 溶液，加热）→ 溶液A →（通入过量 CO₂）→ 沉淀B →（灼烧）→ Al₂O₃

（1）写出 Al₂O₃ 与 NaOH 溶液反应的化学方程式。

（2）溶液A 中含有的主要溶质是哪些？写出 SiO₂ 与 NaOH 反应的离子方程式。

（3）通入过量 CO₂ 生成沉淀B的化学方程式，为什么用过量 CO₂ 而不用适量？

（4）该工艺流程中，Fe₂O₃ 如何被除去？

▶ 查看答案与解析
【解析】
(1) Al₂O₃ + 2NaOH = 2NaAlO₂ + H₂O

(2) 溶液A中主要溶质为 NaAlO₂ 和 Na₂SiO₃（Fe₂O₃ 不与 NaOH 反应，过滤除去）。SiO₂ + 2NaOH = Na₂SiO₃ + H₂O，离子方程式：SiO₂ + 2OH⁻ = SiO₃²⁻ + H₂O。

(3) NaAlO₂ + CO₂ + 2H₂O = Al(OH)₃↓ + NaHCO₃（过量 CO₂ 生成 NaHCO₃）。用过量 CO₂ 是因为 CO₂ 为弱酸酐，与 NaAlO₂ 反应只能生成 Al(OH)₃ 而不会溶解它，且能确保 AlO₂⁻ 完全转化为 Al(OH)₃。若用适量，部分 AlO₂⁻ 可能未完全反应。

(4) Fe₂O₃ 不溶于 NaOH 溶液，在第一步过滤中作为滤渣被除去。

16．（16分）某实验小组用如下装置探究 Fe²⁺ 与 Fe³⁺ 的相互转化及性质。

实验Ⅰ：取少量 FeCl₂ 溶液于试管中，加入 KSCN 溶液，溶液不变色。再加入 H₂O₂ 溶液，溶液变红色。

实验Ⅱ：取少量 FeCl₃ 溶液于试管中，加入 KSCN 溶液，溶液变红色。再加入铁粉，振荡后红色褪去。

（1）实验Ⅰ中加入 H₂O₂ 后溶液变红的原因是什么？写出相关的离子方程式。

（2）实验Ⅱ中红色褪去的原因是什么？写出相关反应的化学方程式和离子方程式。

（3）请设计一个实验方案证明实验Ⅰ中 Fe²⁺ 已完全被氧化为 Fe³⁺（写出操作、现象和结论）。

（4）从氧化还原的角度解释：为什么实验室中 FeCl₂ 溶液中常加入少量铁粉和稀盐酸？

▶ 查看答案与解析
【解析】
(1) H₂O₂ 将 Fe²⁺ 氧化为 Fe³⁺：2Fe²⁺ + H₂O₂ + 2H⁺ = 2Fe³⁺ + 2H₂O。Fe³⁺ 与 SCN⁻ 反应生成红色 Fe(SCN)₃，故溶液变红。

(2) 铁粉将 Fe³⁺ 还原为 Fe²⁺，Fe³⁺ 浓度降低，Fe(SCN)₃ 平衡逆向移动，红色褪去。化学方程式：Fe + 2FeCl₃ = 3FeCl₂。离子方程式：Fe + 2Fe³⁺ = 3Fe²⁺。

(3) 取实验Ⅰ中红色褪去后的溶液少许，加入铁氰化钾 K₃[Fe(CN)₆] 溶液，若不产生蓝色沉淀，则证明 Fe²⁺ 已完全被氧化为 Fe³⁺。

(4) 铁粉可防止 Fe²⁺ 被空气中 O₂ 氧化为 Fe³⁺（2Fe²⁺ + O₂ + 4H⁺ = 2Fe³⁺ + 2H₂O），因 Fe 可将 Fe³⁺ 还原回 Fe²⁺。稀盐酸可抑制 Fe²⁺ 和 Fe³⁺ 的水解（Fe²⁺ + 2H₂O ⇌ Fe(OH)₂ + 2H⁺），保持溶液酸性防止沉淀生成。

17．（14分）已知反应：CO(g) + H₂O(g) ⇌ CO₂(g) + H₂(g)　ΔH = -41.2 kJ/mol。在某温度下，将 1 mol CO 和 3 mol H₂O 充入一个体积为 2 L 的密闭容器中，达到平衡时 CO 的转化率为 80%。

（1）求该温度下反应的平衡常数 K。

（2）若在相同条件下，向该平衡体系中再充入 1 mol CO 和 1 mol H₂，平衡如何移动？请说明理由。

（3）画出该反应在其他条件不变时，不同温度下 CO 的平衡转化率随温度变化的趋势图（定性描述）。

▶ 查看答案与解析
【解析】
(1) 初始：n(CO)=1, n(H₂O)=3, n(CO₂)=0, n(H₂)=0（mol），V=2L。
平衡时 CO 转化率 80%→消耗 0.8 mol CO。
n(CO)=0.2, n(H₂O)=2.2, n(CO₂)=0.8, n(H₂)=0.8（mol）
各物质浓度：c(CO)=0.1, c(H₂O)=1.1, c(CO₂)=0.4, c(H₂)=0.4（mol/L）
K = c(CO₂)·c(H₂)/[c(CO)·c(H₂O)] = (0.4×0.4)/(0.1×1.1) = 0.16/0.11 ≈ 1.455

(2) 再充入 1 mol CO 和 1 mol H₂ 后：n(CO)=1.2, n(H₂O)=2.2, n(CO₂)=0.8, n(H₂)=1.8（mol）
Q = (0.8×1.8)/(1.2×2.2) = 1.44/2.64 ≈ 0.545 < K ≈ 1.455
Q < K，平衡正向移动。

(3) 该反应为放热反应（ΔH < 0），随温度升高，平衡逆向移动，CO 平衡转化率降低。图像为从左上向右下倾斜的曲线（转化率随温度升高而减小）。

18．（14分）某有机化合物 A（分子式为 C₈H₈O₂）有如下性质：

① A 能与 NaOH 溶液反应生成 B 和 C（均为有机物）。

② B 的分子式为 C₇H₆O₂，能与 NaOH 反应，也能与 NaHCO₃ 反应放出 CO₂。

③ C 的分子式为 CH₄O，即 CH₃OH。

④ B 与 FeCl₃ 溶液不发生显色反应。

（1）A 的结构简式为__________，其含有的官能团名称为__________。

（2）B 的结构简式为__________，写出 B 与 NaHCO₃ 反应的化学方程式。

（3）A 在 NaOH 溶液中加热发生反应的化学方程式，并指出该反应的类型。

（4）写出满足分子式为 C₈H₈O₂ 且属于酯类的同分异构体的结构简式（写出3种）。

▶ 查看答案与解析
【解析】
(1) A 的分子式 C₈H₈O₂，不饱和度 = (2×8+2-8)/2 = 5。水解生成 B(C₇H₆O₂) 和 CH₃OH → A 为酯。B 能与 NaHCO₃ 反应→含 —COOH；B 与 FeCl₃ 不显色→不含酚羟基。C₇H₆O₂ 有一个苯环 + 一个 —COOH，即 B 为苯甲酸（C₆H₅COOH）。A 为苯甲酸甲酯 C₆H₅COOCH₃。含有的官能团：酯基（—COO—）。

(2) B 为苯甲酸：C₆H₅COOH。与 NaHCO₃ 反应：C₆H₅COOH + NaHCO₃ = C₆H₅COONa + H₂O + CO₂↑

(3) C₆H₅COOCH₃ + NaOH →(加热) C₆H₅COONa + CH₃OH。反应类型：水解反应（取代反应）。

(4) C₈H₈O₂ 的酯类同分异构体：
① C₆H₅COOCH₃（苯甲酸甲酯）
② C₆H₅CH₂OOCH（甲酸苄酯，HCOOCH₂C₆H₅）
③ CH₃COOC₆H₅（乙酸苯酯）
④ CH₃C₆H₄OOCH（甲酸甲苯酯）等。任选3种即可。
