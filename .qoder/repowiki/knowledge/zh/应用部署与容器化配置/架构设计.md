- 采用多阶段部署架构：`Dockerfile` 定义 Node.js 22 生产镜像，通过 `docker-compose.yml` 编排服务、环境变量及持久化卷。
- 集成系统级守护：`deploy/uibe-graphrag.service` 配置 systemd 单元，用于在 Linux 宿主机上以非 root 用户运行 Python FastAPI/uvicorn 服务，并实施资源限制（内存/CPU）。
- 依赖管理分离：Node.js 依赖通过 `npm ci` 安装，Python 服务依赖虚拟环境路径，两者通过不同的部署入口点（`server.js` vs `uvicorn`）启动。
