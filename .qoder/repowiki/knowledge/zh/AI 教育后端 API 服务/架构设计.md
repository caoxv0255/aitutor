- **入口与路由**：`server.js` 作为应用入口，集中注册所有业务路由（如 `/api/login`, `/api/questions`），并挂载全局中间件。
- **数据持久层**：`api/db.js` 封装 SQLite 连接与初始化逻辑，通过 `getDb()` 单例模式管理数据库实例，并在启动时自动执行建表与种子数据同步。
- **安全与中间件**：`api/middleware/security.js` 实现 XSS 过滤、CSRF 防护及安全头设置；`api/auth.js` 提供 JWT 校验中间件，保护受控接口。
- **异步任务处理**：采用生产者-消费者模式，`api/tasks.js` 负责接收图像识别请求并存入队列，`api/taskWorker.js` 在后台轮询处理并调用 LLM 接口。
- **外部服务代理**：`api/graphrag.js` 作为网关转发 GraphRAG 检索请求，`api/proxy.js` 处理其他第三方 API 调用。
- **工具支撑**：`api/utils/` 目录提供统一的响应格式化 (`response.js`)、LLM 输出解析 (`llmParser.js`) 及输入校验逻辑。
